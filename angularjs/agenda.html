Promises in AngularJS

  • Integration with digest cycle
  • Testing
    In short: necessity to call $rootScope.$digest(), same as in a situation when an event comes from outside of angular.
  • Deferred
    In other words: builder for promises.
    Note: Deprecated in pure JS. obsolete API: Promise.defer() Polyfil and documentation can be found here
  • Unwrapping promises: legacy issue
    First versions of AngularJS (<1.2.x) unwrapped promises into values in html templates.
  • Notify (external example)
    Misconception: it does not call itself after each then, it can be called only by deferred or in constructor
  • All, race
    Other useful method: allSettled -waits until all promises will be settled (either resolved or rejected) and puts result or error codes in array.
    Notice: It will also work in the same way in Pure JS (Author's whim to implement it in AngularJS :P -good usage of $timeout)
